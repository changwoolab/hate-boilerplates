# hate-boilerplates

My utilities for avoiding boilerplates

## Installation

```
cargo install hbp
```

or

```
brew tap changwoolab/hbp
brew install hbp
```

## Usage

```
hbp [command]
```

### Example

```
hbp rnd
```

## Commands

| command | explanation                                             |
| ------- | ------------------------------------------------------- |
| `rnd`   | Shut down the running React Native Debugger and Restart |
