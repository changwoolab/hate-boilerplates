# hate-boilerplates

My utilities for avoiding boilerplates

## Installation

```
cargo install hbp
```

or

```
brew tap changwoolab/hbp
brew install hbp
```

## Usage

```
hbp [command]
```

### Example

```
hbp rn-debugger
```

## Commands

| command       | explanation                                             |
| ------------- | ------------------------------------------------------- |
| `rn-debugger` | Shut down the running React Native Debugger and Restart |
